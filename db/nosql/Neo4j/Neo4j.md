Neo4j - графовая база данных. Минимальным значением в графовых база данных является `узел`. Использует язык запросов Cypher.
Пример создания узла:
```cypher
CREATE (S1:Sotrudnik {TabNom: "1234", Familia: "Иванов", Imya: "Иван", Otchestvo: "Иванович", DataRozdenia: "31.05.1980", Dolznost: "доцент", DataKonkursa: "31.07.023", Telefon: "8-9166405859", Adress: "г.Москва, Нежинская, 10, 56"})
```
